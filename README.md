# GEMINI-AETHER 🌌

**GEMINI-AETHER** es una suite de herramientas de automatización desarrolladas en Python, diseñadas para optimizar tareas repetitivas mediante la manipulación lógica de periféricos (teclado y ratón).

## 🚀 Descripción
Este proyecto centraliza más de 50 scripts funcionales que permiten automatizar flujos de trabajo, desde macros sencillas hasta secuencias complejas de interacción con la interfaz de usuario. El objetivo es proporcionar una base sólida y personalizable para la automatización de tareas en entornos de escritorio.

## 🛠️ Tecnologías utilizadas
*   **Lenguaje:** Python 3.x
*   **Librerías principales:** 
    *   `PyAutoGUI` - Control de ratón y teclado.
    *   `Pynput` - Monitoreo de entradas de usuario.
    *   `Time` - Gestión de retardos y precisión.

## 📋 Características
*   **Automatización de Periféricos:** Control preciso de coordenadas y pulsaciones de teclas.
*   **Modularidad:** Scripts independientes organizados por función.
*   **Ligero:** Bajo consumo de recursos y ejecución rápida.
